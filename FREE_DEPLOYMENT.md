# FREE DEPLOYMENT OPTIONS

## 1. **Vercel** (Frontend) - FREE
- Go to https://vercel.com
- Import Git Repository: `anveshap103-tomato/QURA_UI`
- Click Deploy
- Live in 2 minutes!

## 2. **PythonAnywhere** (Backend) - FREE
- Go to https://www.pythonanywhere.com
- Sign up free account
- Upload backend files
- Run: `pip install -r requirements.txt`
- Configure WSGI

## 3. **Netlify** (Frontend) - FREE
- Go to https://netlify.com
- Import from GitHub: `QURA_UI`
- Auto-deploys using `netlify.toml`

## 4. **Render.com** - FREE (No card needed)
- Go to https://render.com
- Sign up with GitHub
- New → Web Service
- Connect `QURA_UI` repo
- Select "Free" plan
- Deploy!

## 5. **Railway.app** - FREE $5 credit
- Go to https://railway.app
- Sign up with GitHub
- New Project → Deploy from GitHub
- Select `QURA_UI`
- Free $5/month credit

## RECOMMENDED: Split Deployment
**Frontend (Vercel)** + **Backend (Render Free)**
- Both 100% free
- No credit card required
- Best performance
